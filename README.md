# Islara Dashboard - Gestión de Obra

Un dashboard moderno y premium para la gestión de proyectos de construcción, seguimiento financiero y bitácora de obra.

![Dashboard Preview](https://via.placeholder.com/800x400?text=Islara+Dashboard+Preview)

## ✨ Características

-   **Cronograma Interactivo**: Seguimiento de tareas por fases, estados y presupuesto.
-   **Finanzas en Tiempo Real**: Visualización de Presupuesto vs Gasto Real (Curva S).
-   **Bitácora de Obra**: Registro diario de clima, notas y gastos con persistencia.
-   **Reportes Profesionales**: Exportación de datos a CSV y Bitácora a PDF.
-   **Personalización**: Configura el nombre y subtítulo de tu proyecto.
-   **Datos Locales**: Todo se guarda en tu navegador. Privacidad total.

## 🚀 Instalación y Uso

Este proyecto utiliza **Next.js**.

1.  **Instalar dependencias**:
    ```bash
    npm install
    ```

2.  **Iniciar servidor de desarrollo**:
    ```bash
    npm run dev
    ```
    Visita `http://localhost:3000` en tu navegador.

## 🛠️ Tecnologías

-   **Frontend**: React, Next.js, TypeScript.
-   **Estilos**: Tailwind CSS, Lucide React (Iconos).
-   **Animaciones**: Framer Motion.
-   **Gráficos**: Recharts.
-   **Utilidades**: date-fns (Fechas), jsPDF (Reportes).
